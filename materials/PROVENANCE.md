# CS50 C Accelerator Provenance

This site is grounded in CS50x C-related Weeks 1-5 only. Teaching sections follow CS50's order: lecture material, official shorts, section-style guided practice, then programming assignments. Extra exercises are labeled as practice steps and are the only intentionally supplemental material.

## Coverage Map

- Week 1 covers CS50's C start: VS Code for CS50, Linux, source code, hello world, headers, manual pages, input, types, format codes, variables, conditionals, operators, loops, functions, correctness, design, style, Mario, Hello, Cash, and Credit.
- Week 2 covers arrays, strings, string length, debugging, compiling, command-line arguments, exit status, Scrabble, Readability, and Caesar.
- Week 3 covers linear search, binary search, running time, `search.c`, `phonebook.c`, structs, selection sort, bubble sort, recursion, merge sort, Sort, Plurality, Runoff, and Tideman.
- Week 4 covers pixel art, hexadecimal, memory, pointers, strings as pointers, pointer arithmetic, string comparison, copying, `malloc`, Valgrind, garbage values, Pointer Fun with Binky, swapping, overflow, `scanf`, file I/O, Volume, Filter, and Recover.
- Week 5 covers Jack Learns the Facts, data structures, queues, stacks, arrays, linked lists, trees, hash tables, tries, Inheritance, and Speller.

## Evidence UI

Each accelerator lesson has collapsed Evidence and Sources toggles. They are intentionally small so they do not interrupt learning, but they remain available when you want to audit where a teaching point came from.
